# EazyBids

A web application built to help transit employees sort, filter, and analyze work assignments and bid for shifts. Designed originally for train operators, this tool makes it easy to compare schedules, find preferred shifts, and save customized lists for future reference.

[View the demo](https://zack-young-ideas.github.io/eazybids/)

## Getting Started

### Prerequisites
- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/zack-young-ideas/eazybids.git
cd eazybids
npm install
```

### Run the Development Server

```bash
npm run dev
```

Then open http://localhost:3000.

### Build for Production

```bash
npm run build
npm start
```
